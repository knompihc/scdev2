This repository contains code for testing purpose of streetcomm 2.0
It will be used as follows:
	1) http server is running at 8004 can be changed in hcserver.go
	2) Start server : go run hcserver.go
	3) Start gateway client : go run client.go localhost:13004
	4) open localhost:8004
	5) put some data and hit OK
	6) wait and see response
